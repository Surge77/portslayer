# Security Policy

## Supported Versions

Security fixes are provided for the latest published version of `portslayer-cli`.

## Reporting a Vulnerability

Please do not open public issues for security problems.

Instead, report privately by email:

- `tejasdeshmane66@gmail.com`

Include:

- Description of the vulnerability
- Impact and affected versions
- Reproduction steps or proof of concept
- Suggested fix (if available)

## Response Timeline

- Initial acknowledgement: within 72 hours
- Status update: within 7 days
- Fix and disclosure timeline: based on severity and exploitability

## Disclosure Policy

- We follow responsible disclosure.
- Credit can be provided in release notes if requested.
